## Vulnerability index — quick reference

```
vulnerability_log2ratio = log2(metabolic_load + eps) - log2(proteostasis_capacity + eps)
```

- **metabolic_load** panel: OXPHOS + TCA + ion pumps + Ser/Gly/Gln biosynthesis + ribosomes + splicing factors / DEAD-box helicases. ~150 genes. See [`_gene_sets.py`](_gene_sets.py).
- **proteostasis_capacity** panel: HSP70/90 + TRiC + proteasome + ubiquitin + autophagy + lysosomal + ERAD/UPR. ~80 genes.

### Scoring method differs between Fig 2i–j and Fig 2k–l

| Panel | Method | Why |
| --- | --- | --- |
| Fig 2i–j (public bulk + scRNA-seq) | `geneset_mean` (mean of log-normalised expression per panel) | Preserves absolute magnitude on a comparable scale across datasets |
| Fig 2k–l (in-house Smart-seq3 VHC) | `sc.tl.score_genes` (background-subtracted; can be negative) | Better dynamic range for separating fine VHC subtypes within one dataset |

Reviewer should be aware of this asymmetry — both implementations are documented inline in each notebook.

## Notes on pipeline scope

These notebooks cover the **final plotting / analysis** step. Upstream steps (QC, integration, contamination filtering, pySCENIC GRN inference) are described in the manuscript Methods and are not bundled here.

- Contamination filtering uses a 4-axis blacklist (stress / mito-ribo / glia / metabolic) plus a covariate Spearman ρ check.
- pySCENIC outputs (`scenic_output_*.loom`) are produced via the standard GRNBoost2 → cisTarget → AUCell pipeline.
- IHC/OHC labels on public scRNA-seq datasets are derived in-notebook by leiden re-clustering + `Slc17a8` (IHC) / `Slc26a5` (OHC) marker check. Cluster numbering is not stable across reruns — see the in-notebook warning at the start of each Fig 2j section.
